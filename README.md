# **More Anvils**
Anvils of every metal
---
<br>

## Installation:
- Install Fabric by following the [Fabric Installation Guide](https://wiki.fabricmc.net/install).
- Download the latest version of More Anvils from the [releases page](https://github.com/OrcaCrafter/MoreAnvils/releases).
- Place the **`more-anvils.jar`** file into the **`mods`** folder of your Fabric installation.
